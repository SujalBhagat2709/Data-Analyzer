# Data Analyzer

## Overview

A simple tool to analyze numerical data and CSV files.

---

## Features

* Calculate sum, count, mean
* Analyze numeric columns from CSV

---

## Files

* `stats.py` → basic statistics
* `csv_analyzer.py` → analyze CSV data

---

## Usage

```bash id="u6fj3j"
python stats.py
python csv_analyzer.py
```

---

## Example Output

```id="xtxt7w"
{'sum': 100, 'count': 4, 'mean': 25.0}
```
